01 — Project Overview

A digital presence built for Vietnam's most ambitious architecture firm.

SUNJIN Vietnam commands a reputation for landmark commercial and residential architecture across Vietnam. The website was engineered to match that standing — a dark editorial aesthetic that lets the architecture breathe, with precision typography that signals authority without shouting.

Seven interconnected pages cover every touchpoint: portfolio, thought leadership, culture, and talent acquisition — each designed with the same visual grammar, each optimised for its specific job.

06 — Projects Page

Projects

project.html

The portfolio. An editorial grid of 9 projects arranged in an intentionally asymmetric pattern — full-width, half-half, thirds — creating visual rhythm as the user scrolls. A sticky filter bar lets visitors narrow by typology. Hairline separators between cards are achieved by a 1px gap on a white container, giving precision without hard borders.

Sticky filter bar — categories scroll with page, active item underlined in white
Editorial grid — 4 rows: 1×full, 2×split, 3×thirds, 2×split with varied heights
Hairline dividers — gap-px on a white/12% container creates 1px hairline separators
Equal-height rows — explicit height (vh) per row instead of aspect-ratio
Card images — position:absolute; inset:0 fills entire card regardless of ratio
Hover: image zooms 5%, project details fade in from bottom
JS filter — hides non-matching cards with smooth opacity transition

Blog

blog.html

Thought leadership and editorial content across four categories: News, Press Coverage, Ideas, and Community Responsibility. Eight articles arranged in an asymmetric editorial grid — the layout deliberately avoids uniform card sizes to create a magazine-like visual hierarchy. On hover, article excerpts and read-more links emerge from below the image.

4 categories — News · Press Coverage · Ideas · Community Responsibility
Editorial grid — 4 rows with varying column splits (full / 7+5 / 3×equal / 5+7)
Card hover — image zooms, overlay darkens, excerpt fades up, arrow appears
Category pill — glass-blur tag (backdrop-filter) positioned over image top-left
JS filter — smooth hide/show by data-cat attribute matching
Matching row heights — same explicit height per row for visual alignment

08 — Blog Detail Page

Blog Detail

blog-detail.html

The reading experience. A full-screen dark hero — category pill, back link, title, meta bar — gives the article a cinematic open. Immediately below the hero, a 65vh full-bleed banner image sets the visual tone before the article body begins. Text is full-width with generous padding rather than a narrow column, keeping the editorial feel wide and spacious.

Reading progress bar — fixed 2px line at viewport top, width tracks scrollY / scrollHeight
Full-screen hero — category pill, ← back link, large title, author + date meta bar
65vh hero banner image — full-bleed between hero and article body
Article body — full-width with px-6 md:px-16 lg:px-28 xl:px-40 padding
Full-bleed in-article image — negative margins break out of text padding
Pull quote — left border accent, italic Jost, visually distinct from body text
Related articles — 3-column grid of article cards at bottom

09 — Careers Page

sunjinvietnam.vn / careers.html

Careers

careers.html

Talent acquisition positioned as a brand moment. A dark hero opens the page, followed by a cinematic full-width banner photo and a culture strip of three candid studio images. The job listing section shifts to a deliberately white background — a visual signal of a different mode — with pill-style multi-axis filters (Department, Location, Level) and a clean tabular list of open positions.

Dark hero + 58vh full-bleed banner photo — same visual grammar as other pages
3-photo culture strip — hover reveals overlay text, communicates studio life
White listing section — deliberate break from dark theme signals different context
Multi-axis filter — 3 sets of pill chips (Dept / Location / Level), combined logic
Job rows — title + meta + arrow, hover highlights full row
Second culture strip — 3 more photos below listing with overlay hover text
JS filter logic — activeFilters object, all 3 axes combined with AND logic

10 — Careers Detail Page

Careers Detail

careers-detail.html

A split-screen detail page that breaks from the standard scrolling layout. The left panel is fixed — a full-height studio image with the job title and a CLOSE button overlay. The right panel scrolls independently through the job's details: a 2×2 meta grid (Level, Salary, Deadline, Division), expandable accordion sections, and an application form with CV upload.

Split-screen layout — position:fixed; left:0; width:50% panel + margin-left:50% scrollable panel
Meta grid — 2×2 cards: Level · Salary · Deadline · Division/Location
Accordion sections — max-height CSS transition, Job Description open by default
Application form — Name, Email, Phone, Cover Note, CV upload field
CV upload — custom styled file input with filename display, updateUploadLabel()
Submit success — button transforms to checkmark confirmation state
Mobile — left panel becomes relative 60vh block, right stacks below
